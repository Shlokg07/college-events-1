## 💡 Common Use Cases

### "I want to show a success message with animation"
```javascript
const msg = document.createElement('div');
msg.textContent = '✅ Success!';
msg.className = 'success-message';
document.body.appendChild(msg);
AnimationManager.success(msg);
```

### "I want to fade out an element and remove it"
```javascript
AnimationManager.fadeOut(element, 300);
setTimeout(() => element.remove(), 300);
```

### "I want to animate all cards when content loads"
```javascript
AnimationManager.animateCards();
```

### "I want custom animation speed"
```javascript
AnimationManager.fadeIn(element, 1000); // 1 second instead of default
```
